#  SREE - Virtual Assistant (Python)

SREE is a voice-based **Virtual Assistant** built using Python.  
It can listen to your commands, speak responses, open apps/websites, search Google/YouTube, type in Notepad, send WhatsApp messages, take screenshots, control volume, and more.
